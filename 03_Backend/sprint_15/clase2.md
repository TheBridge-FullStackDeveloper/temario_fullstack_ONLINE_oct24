    ```javascript

    module.exports = router;
    ```

    ```javascript

    const connectDB = require("./config/dbConfig");
    const userRoutes = require("./routes/userRoutes");

    app.use(express.json());
    app.use("/api", userRoutes);
    
    connectDB(); // Conexi√≥n a la base de datos
    
    module.exports = app;
    ```
    
6. **`index.js`**  
    Inicia el servidor:
    
    ```javascript

    const PORT = process.env.PORT || 8080;
    
    app.listen(PORT, () => {
      console.log(`Server running on port ${PORT}`);
    });
    ```

2. La ruta en `userRoutes.js` dirige la solicitud al controlador `getUsers` en `userController.js`.

5. La respuesta se env√≠a al cliente en formato JSON.

#### **1. `src/utils/validationUtils.js`**

  const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return emailRegex.test(email);

#### **2. `src/middlewares/errorHandler.js`**

  });

```

---

```

## **1. Flujo de trabajo: Estructura b√°sica (sin `services` ni `utils`)**

---

### **Resumen del flujo b√°sico:**

1. Cliente ‚Üí **Ruta (`routes`)** ‚Üí **Controlador (`controllers`)** ‚Üí **Modelo (`models`)** ‚Üí Base de datos.
